Nihar Construction and Buildings is a dynamic and growing company in the construction and real estate sector, dedicated to delivering high-quality residential and commercial projects. The company focuses on building strong, durable, and aesthetically appealing structures that meet modern standards and customer expectations.
# 🏗️ Nihar Construction and Buildings

## 📌 Project Overview

**Nihar Construction and Buildings** is a web-based project designed to represent a construction company’s services, portfolio, and company profile. The project showcases modern construction practices, company values, and project offerings in a simple and user-friendly interface.

This project is ideal for academic purposes (B.Tech) and demonstrates basic web development using HTML and CSS.

---

## 🎯 Objectives

* To design a professional website for a construction company
* To showcase services like residential, commercial, and interior work
* To create a clean and responsive layout using HTML & CSS
* To improve frontend development skills

---

## 🛠️ Technologies Used

* **HTML5** – Structure of the website
* **CSS3** – Styling and layout
* **Basic Web Design Principles**
